== Description ==

[Homepage](https://accesspressthemes.com/) | [Documentation](https://accesspressthemes.com/documentation/wp-popup-lite/) | [Support](https://accesspressthemes.com/support) | [Demo](http://demo.accesspressthemes.com/wordpress-plugins/wp-popup-lite)

<strong>WP Popup Lite</strong> is a FREE Popup plugin for WordPress. You can use this to add popups to your site with custom images or messages with different effects.

### Features ###
- <strong>Unlimited Popup</strong>
- <strong>13 inbuild themes</strong>
- <strong>Multipurpose Theme</strong> - for multi usability such as subscription, contact form, product display and coupon display.
- <strong>Custom Popup Theme</strong> - to create your own custom popup if you want with following elements editable.
	* Text
	* Image
	* Pdf
	* Youtube video embed
- <strong>Contact Form Log</strong> - view all the data sent from the contact form popup and manage form data
- <strong>Admin Alert</strong> - Enable/ Disable admin alert on form submission
- <strong>Mailchimp Integration.</strong>
- <strong>Basic Features</strong>
    * Animation for popup
    * Option to show popup either always or just once
    * Delay popup display
    * Auto Close popup
    * Overlay Configuration Options
    * Close Button
    * Background color/image option
    * Border Options
- <strong>User Friendly Interface</strong>
- <strong>Responsive Ready</strong>
- <strong>Translation Ready</strong>

== Installation ==
1. Unzip wp-popup-lite.zip
1. Upload all the files to the /wp-content/plugins/wp-popup-lite
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For creating new popup, click on the Add New Popup submenu on right side of the screen.
